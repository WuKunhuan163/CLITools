# KIMI

Kimi AI assistant API integration

**Purpose**: Access Kimi AI capabilities for long-context understanding and generation.

## Capabilities

- chat
- long-context
- document-analysis

## Usage

```bash
KIMI status          # Show tool status
KIMI config <k> <v>  # Set configuration
KIMI setup           # Install dependencies
```

## Environment Variables

- `KIMI_API_KEY`
