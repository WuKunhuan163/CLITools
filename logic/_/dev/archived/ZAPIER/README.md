# ZAPIER

Workflow automation via Zapier MCP

**Purpose**: Access 8000+ apps and 40000+ actions through Zapier's automation platform.

## Capabilities

- trigger-zap
- list-actions
- execute-action

## Usage

```bash
ZAPIER status          # Show tool status
ZAPIER config <k> <v>  # Set configuration
ZAPIER setup           # Install dependencies
```

## Environment Variables

- `ZAPIER_MCP_URL`

## API Key

Obtain credentials at: https://actions.zapier.com/settings/mcp
