# SUNO

AI music generation via Suno API

**Purpose**: Generate AI music, lyrics, and audio content using Suno models.

## Capabilities

- music-generation
- lyrics-generation
- song-extension
- cover-remix

## Usage

```bash
SUNO status          # Show tool status
SUNO config <k> <v>  # Set configuration
SUNO setup           # Install dependencies
```

## Environment Variables

- `ACEDATACLOUD_API_TOKEN`

## API Key

Obtain credentials at: https://platform.acedatacloud.com
