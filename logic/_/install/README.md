# logic/_/install/ — Install Command Resources

Binary resources and archived tool snapshots used by the install/setup system.

## Structure

| Directory | Contents | Git Tracking |
|-----------|----------|-------------|
| `archived/` | Tool snapshots from `tool/` branch (auto-generated by `TOOL dev sync`) | `tool` branch only |
| `resource/` | Binary assets (Python builds, fonts, etc.) | `tool` branch only |

## How It Works

1. `TOOL dev sync` runs `align_tool` which archives all tools from `tool/` into `archived/`
2. Binary resources (Python builds, font files) are stored in `resource/<TOOL_NAME>/data/`
3. `TOOL --install <name>` first checks `tool/` branch, then falls back to `archived/`

## Branch Behavior

- **dev branch**: This directory is gitignored (no binary bloat)
- **tool branch**: Tracked via `git add -f` (preserves snapshots and resources)
- **main branch**: Not present (production-only framework)
