# logic/_/install/ — Agent Guide

Install command resources. Two subdirectories:

- `archived/` — Snapshots of every tool, auto-generated by `TOOL dev sync`. 
  Used as fallback when installing tools not on current branch.
- `resource/` — Binary assets per tool (Python builds, fonts). 
  Referenced by `RESOURCE_ROOT` in tool code.

Key paths:
- Python builds: `resource/PYTHON/data/install/<version>/`
- Font files: `resource/FONT/data/install/<name>/`
- Archived tool: `archived/<TOOL_NAME>/`

These directories are only tracked on the `tool` git branch.
On `dev`, they're gitignored. Code references use Path-based constants.
