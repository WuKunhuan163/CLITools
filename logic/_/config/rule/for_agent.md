# logic/config/rule — Agent Reference

## manager.py — RuleManager

Generates the `.cursor/rules/AITerminalTools.mdc` file from:
- Tool registry (`tool.json`)
- Tool metadata (descriptions, purposes)
- Available skills index
- Development guidelines

Uses `get_translation()` for localized rule labels.

## Gotchas

1. **Auto-generated file**: `.cursor/rules/AITerminalTools.mdc` is regenerated by RuleManager. Manual edits will be overwritten.
2. **Depends on tool.json**: Rule content reflects current tool registry. Add new tools to `tool.json` before regenerating.
