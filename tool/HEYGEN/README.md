# HEYGEN

AI avatar video generation via HeyGen API

**Purpose**: Create AI avatar videos with voice selection and video generation.

## Capabilities

- avatar-management
- voice-selection
- video-generation

## Usage

```bash
HEYGEN status          # Show tool status
HEYGEN config <k> <v>  # Set configuration
HEYGEN setup           # Install dependencies
```

## Environment Variables

- `HEYGEN_API_KEY`

## API Key

Obtain credentials at: https://docs.heygen.com/docs/heygen-mcp-server

## MCP Backend

Package: `heygen-mcp-server`
