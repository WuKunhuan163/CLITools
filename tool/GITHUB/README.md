# GITHUB

GitHub integration via GitHub MCP

**Purpose**: Manage repositories, issues, pull requests, and code through GitHub API.

## Capabilities

- repositories
- issues
- pull-requests
- code-search
- file-operations

## Usage

```bash
GITHUB status          # Show tool status
GITHUB config <k> <v>  # Set configuration
GITHUB setup           # Install dependencies
```

## Environment Variables

- `GITHUB_PERSONAL_ACCESS_TOKEN`

## API Key

Obtain credentials at: https://github.com/settings/tokens

## MCP Backend

Package: `@modelcontextprotocol/server-github`
