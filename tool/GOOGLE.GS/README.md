# GOOGLE.GS — Google Scholar

## Status: Restricted

Google Scholar's Terms of Service prohibit automated access. This tool's
browser automation has been removed to comply with those terms.

Run `GS` to see available alternatives (Semantic Scholar, OpenAlex, SerpAPI).

## Alternatives

- **Semantic Scholar API** — Free, 100 req/sec, excellent academic coverage
- **OpenAlex API** — Free, open, comprehensive works database
- **SerpAPI** — Paid service providing structured Google Scholar data

See `for_agent.md` for detailed comparison and integration guidance.
