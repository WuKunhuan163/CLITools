# OPENCLAW Skills — Agent Guide

These skills are specific to the OpenClaw self-improvement loop.
For general development skills, navigate to `skills/` at the project root.

## When to use these skills

- **Before any task**: `memory-recall` — check for past lessons
- **After failures**: `experience-accumulation` — record what went wrong
- **When patterns repeat**: `skill-formation` — codify into a skill
- **When stuck on tool bugs**: `tool-resilience` — decision framework
- **When exploring new code**: `source-exploration` — systematic approach
- **When deciding artifact type**: `formulation-guide` — skill vs tool
