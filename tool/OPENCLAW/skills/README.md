# OPENCLAW Skills

Tool-specific skills for the OpenClaw self-improvement loop.

These skills guide agents through OPENCLAW's metacognitive workflows:
experience recording, lesson promotion, skill formation, and tool resilience.

## Skills

| Skill | Purpose |
|---|---|
| experience-accumulation | Recording and promoting lessons into permanent knowledge |
| formulation-guide | Deciding between creating Skills vs Tools |
| memory-recall | Using institutional memory before starting tasks |
| skill-formation | When and how to create new skills from experience |
| source-exploration | Exploring unfamiliar codebases systematically |
| tool-resilience | Handling buggy or missing tools |

## Relationship to Ecosystem Skills

General development skills live in the project-wide `skills/` hierarchy.
These OPENCLAW-specific skills extend the meta-agent philosophy
(`skills/_/meta-agent`) with concrete operational procedures.
