# ASANA

Project management via Asana MCP

**Purpose**: Manage tasks, projects, and workspaces through Asana.

## Capabilities

- list-tasks
- create-task
- update-task
- search-projects

## Usage

```bash
ASANA status          # Show tool status
ASANA config <k> <v>  # Set configuration
ASANA setup           # Install dependencies
```

## Environment Variables

- `ASANA_ACCESS_TOKEN`

## API Key

Obtain credentials at: https://app.asana.com/0/developer-console

## MCP Backend

Package: `@iflow-mcp/mcp-server-asana`
