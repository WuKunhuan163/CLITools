# WHATSAPP

WhatsApp Business API integration

**Purpose**: Send and receive WhatsApp messages via the Business API.

## Capabilities

- send-message
- send-template
- receive-message

## Usage

```bash
WHATSAPP status          # Show tool status
WHATSAPP config <k> <v>  # Set configuration
WHATSAPP setup           # Install dependencies
```

## Environment Variables

- `WHATSAPP_TOKEN`
- `WHATSAPP_PHONE_ID`

## API Key

Obtain credentials at: https://developers.facebook.com/docs/whatsapp
