# XMIND

Mind mapping and brainstorming tool

**Purpose**: Create, edit, and export mind maps for project planning and brainstorming.

## Capabilities

- create-mindmap
- export-mindmap
- import-mindmap

## Usage

```bash
XMIND status          # Show tool status
XMIND config <k> <v>  # Set configuration
XMIND setup           # Install dependencies
```
