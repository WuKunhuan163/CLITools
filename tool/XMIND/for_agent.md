# XMIND - Agent Guide

Mind mapping and brainstorming tool

## Quick Reference

```bash
XMIND status          # Check status and capabilities
XMIND config <k> <v>  # Set API credentials
XMIND setup           # Install dependencies and configure
```

## Capabilities

- create-mindmap
- export-mindmap
- import-mindmap


## Notes

- Run `XMIND setup` before first use to install MCP dependencies.
- API credentials are stored in `tool/XMIND/data/config.json`.
- Use `XMIND --json` for machine-readable output.
