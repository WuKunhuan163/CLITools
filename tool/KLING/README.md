# KLING

AI video generation via Kling API

**Purpose**: Generate AI videos, images, and lip-sync content using Kling AI models.

## Capabilities

- text-to-video
- image-to-video
- lip-sync
- image-generation
- virtual-try-on

## Usage

```bash
KLING status          # Show tool status
KLING config <k> <v>  # Set configuration
KLING setup           # Install dependencies
```

## Environment Variables

- `KLING_ACCESS_KEY`
- `KLING_SECRET_KEY`

## API Key

Obtain credentials at: https://app.klingai.com/global/dev/api-key

## MCP Backend

Package: `mcp-kling`
