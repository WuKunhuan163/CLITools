# LINEAR

Product development system via Linear MCP

**Purpose**: Manage issues, projects, and initiatives in Linear.

## Capabilities

- list-issues
- create-issue
- update-issue
- search

## Usage

```bash
LINEAR status          # Show tool status
LINEAR config <k> <v>  # Set configuration
LINEAR setup           # Install dependencies
```

## Environment Variables

- `LINEAR_API_KEY`

## API Key

Obtain credentials at: https://linear.app/settings/api

## MCP Backend

Package: `@linear/mcp-server`
