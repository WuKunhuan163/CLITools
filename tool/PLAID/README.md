# PLAID

Financial data integration via Plaid API

**Purpose**: Connect bank accounts, retrieve transactions, and verify identities.

## Capabilities

- link-accounts
- transactions
- balances
- identity-verification

## Usage

```bash
PLAID status          # Show tool status
PLAID config <k> <v>  # Set configuration
PLAID setup           # Install dependencies
```

## Environment Variables

- `PLAID_CLIENT_ID`
- `PLAID_SECRET`

## API Key

Obtain credentials at: https://dashboard.plaid.com/developers/keys
