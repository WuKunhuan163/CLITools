# MIDJOURNEY

AI image generation via Midjourney API

**Purpose**: Generate and transform images using Midjourney AI models through AceDataCloud.

## Capabilities

- text-to-image
- image-transform
- upscale
- variations
- blend

## Usage

```bash
MIDJOURNEY status          # Show tool status
MIDJOURNEY config <k> <v>  # Set configuration
MIDJOURNEY setup           # Install dependencies
```

## Environment Variables

- `ACEDATACLOUD_API_TOKEN`

## API Key

Obtain credentials at: https://platform.acedatacloud.com

Package: `mcp-midjourney`
