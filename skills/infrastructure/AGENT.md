# skills/infrastructure/ — Agent Guide

Patterns for building reliable runtime systems. Read when implementing caching, logging, progress displays, error handling, or GUI features.

**Display:** `turing-machine-development` for CLI progress bars and multi-stage workflows.

**Error handling:** `error-recovery-patterns` for retry/backoff, `preflight-checks` for validation before risky operations.

**Data management:** `record-cache` for caching, `retention-rotation` for cleanup policies, `session-debug-log` for logging.

**GUI:** `unified-gui-development` for HTML+CLI synchronization, `setup-tutorial-creation` for wizard interfaces.

**Going up:** `skills/` for the full hierarchy. `skills/_/symmetric-design/` for the structural principle behind these patterns.
