# skills/quality/ — Agent Guide

Skills for verifying and maintaining code quality. Use before committing, during reviews, or when auditing.

**Automated checks:** `code-quality-review` describes what `TOOL --audit` does. Run it regularly.

**Before creating new code:** Check `avoid-duplicate-implementations` to ensure you're not rebuilding something that exists.

**Structural reviews:** `directory-audit-strategy` for verifying directory symmetry.

**Going up:** `skills/` for the full hierarchy. `skills/_/modularization/` for the underlying principle.
