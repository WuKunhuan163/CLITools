# skills/workflow/ — Agent Guide

How agents operate within this ecosystem. Start here when you're a new agent or resuming a session.

**First time?** Read `agent-onboarding` immediately — it covers the bootstrap protocol.

**Planning a task?** Read `task-orchestration` for decomposition strategies, then check `recipes` for worked examples.

**Self-improvement:** `openclaw` describes the Error → Lesson → Skill → Hook pipeline. `skill-creation-guide` explains when and how to create new skills.

**Going up:** `skills/` for the full hierarchy. `skills/_/meta-agent/` for the philosophy behind these workflows.
