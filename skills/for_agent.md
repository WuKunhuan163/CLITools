# skills/ — Agent Guide

This is the root of the skill dictionary tree. Skills are organized by category, each level narrowing the topic.

**Finding a skill by topic:**
- Building something new? → `development/`
- Verifying code quality? → `quality/`
- Implementing runtime patterns? → `infrastructure/`
- Agent workflow and self-improvement? → `workflow/`
- Browser automation? → `browser/`
- IDE-specific features? → `IDE/`
- Foundational principles? → `_/`

**Finding a skill by name:**
Run `SKILLS show <name>` — it searches across all locations.
