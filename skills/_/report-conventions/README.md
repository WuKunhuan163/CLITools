# Report Conventions Skill

Development report conventions for AITerminalTools. Covers naming (`YYYY-MM-DD_topic.md`), namespacing (`report/<scope>/`), CLI integration, and how to record development insights with minimal overhead.

## Quick Reference

- **Create**: `create_report(scope, topic, content)` from `logic/_/dev/report.py`
- **List**: `list_reports(scope)` — returns all reports in a scope
- **Naming**: `YYYY-MM-DD_topic-slug.md` (auto-generated by `create_report()`)
- **Location**: `report/` at root, with namespace subdirectories

See `SKILL.md` for the full guide.
